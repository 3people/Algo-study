## 스터디 방법
1. 매주 월요일 문제당번이 5개의 문제를 가져옵니다.
2. 해당 주 토요일까지 문제를 풀어 커밋한 뒤 `Pull Requests`를 이용해 문제 풀이방법과 코드를 올립니다.
3. 다음 주 월요일날 실시간 리뷰하기
